## Cloning

### SSH, HTTPS, CLI
```
git clone https://github.com/dyanarzobal/Github-Examples.git
```

## Add
Add changes to staged

```
git add Readme.md
```

## Reset
Reset allows you to move Staged changes to be unstaged. Undo changes in staged.

```
git add .
git reset 
```

## Gitconfig file
It shows what stores on your global config like email,name, editor, etc.
``` 
git config --list
```

setup your name and email
```
git config -- global user.name ""

```
